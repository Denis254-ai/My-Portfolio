> "Code is poetry, design is philosophy."

This portfolio represents a "Dark Futurism" aesthetic, designed to showcase high-level engineering capabilities and a deep understanding of user experience. Built for performance and impact, it features localized content for the Kenyan market and interactive physics-based animations.

## Key Features

- **Swipeable Services Carousel**: A custom-built, snap-scrolling "Product" card system.
- **Interactive Toolkit**: Hover and click-to-reveal interactions detailing specific tech stacks.
- **Physics-Based Animations**: Powered by Framer Motion for a premium, fluid feel.
- **Localized Content**: Tailored testimonials and contact details for regional relevance.
- **Responsive Design**: Flawless execution across all device sizes.

## Quick Start

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Denis254-ai/My-Portfolio.git
    cd My-Portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
